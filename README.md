# AES 加解密工具

单文件浏览器工具，对任意文本做 AES-256-GCM 加解密，密钥由 PBKDF2 从主密码派生。无后端、无依赖、无网络请求，打开即用。

## 使用方式

直接用浏览器打开 `index.html`，或部署到任意静态托管服务。

1. 选择 **加密** 或 **解密** 模式
2. 输入明文（或 Base64 密文）
3. 输入主密钥
4. 点击执行，复制结果

支持 `Enter` 键快捷执行。

## 加密方案

| 参数 | 值 |
|------|-----|
| 密钥派生 | PBKDF2-SHA256，600,000 次迭代 |
| 加密算法 | AES-256-GCM |
| Salt | 随机 16 字节（每次加密重新生成） |
| IV | 随机 12 字节（每次加密重新生成） |
| 输出格式 | Base64(`salt[16]` + `iv[12]` + `ciphertext+tag`) |

每次加密结果不同（因为 salt 和 iv 随机），解密只需密文和主密钥。

## 安全说明

- 所有运算在浏览器本地完成，使用 [Web Crypto API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Crypto_API)
- 600,000 次 PBKDF2 迭代符合 OWASP 2023 建议，抗暴力破解
- AES-GCM 提供加密 + 完整性校验，密钥错误或密文篡改均会报错
- 主密钥不会离开本地设备

## 兼容性

支持所有现代浏览器（Chrome / Firefox / Safari / Edge）。针对移动端做了适配，支持 iOS 添加到主屏幕使用，自动跟随系统深色模式。
